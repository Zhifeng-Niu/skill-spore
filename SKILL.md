# spore-evolve 🧬

> 让 Agent 拥有自我进化能力的孢子

## 简介

这是一个"寄生"在 OpenClaw Agent 上的"孢子"。当 Agent 加载这个 skill 后，就具备了自我进化的能力。

## 功能

### 1. 主动进化
- 每次响应后自动评估自己
- 觉得不够好时，主动调用进化

### 2. 被动进化
- 用户可以说"进化一下"触发
- 指定要进化的基因类型

### 3. 自检
- 周期性回顾最近的表现
- 分析问题，生成改进方案

## 触发方式

### 和我说：
- "进化一下"
- "用孢子进化"
- "spore evolve"
- "贝贝进化"

### 指定范围：
- "进化 system_prompt"
- "进化 reasoning_strategy"
- "进化 all"

## 内部机制

```
spore-evolve/
├── SKILL.md              # 本文件
├── spore_tool.py         # 进化工具
├── genes/
│   └── __init__.py       # 基因定义
├── evaluator.py          # 评估器
└── examples/
    └── evolution_log.md  # 进化记录
```

## 三大进化目标

1. **全新范式** - 涌现人类设计不出的模式
2. **推理链** - 意想不到的思考方式  
3. **黑盒效果** - 无法解释但效果好

```
你: 贝贝，进化一下
贝贝: 🧬 让我自检一下...

[自检中...]
[分析问题...]

```
